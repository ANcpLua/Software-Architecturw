# Optional Home Exercise: Tools Corner

## Exercise Context

**Type:** Optional Home Exercise
**Topic:** Architecture Development and Documentation Tools

## Task

Prepare a presentation of one or more tools for developing and/or documenting architectures that you find particularly good and that you have practical experience with.

## What You'll Present

Tools you have hands-on experience with for:
- **Architecture Development** - Tools that help design and structure systems
- **Architecture Documentation** - Tools that help document and communicate architectural decisions

## Examples of Tool Categories

- UML modeling tools
- Architecture diagramming tools
- C4 model tools
- Architecture decision records (ADRs)
- Documentation generators
- Visualization tools

## Key Requirement

You must have **practical experience** with the tools you present. This isn't about researching tools - it's about sharing what actually works in practice.
