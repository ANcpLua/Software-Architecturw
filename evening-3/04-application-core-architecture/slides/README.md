## Exercise Context

**Exercise ID:** EarlyBird12
**Evening:** 3
**Type:** Home
**PDF:** Page 228 in Architecture Development slides

**Task:** Design application core architecture for the EarlyBird breakfast delivery system.

**Evolution Scenarios to Analyze:**
a. Standing orders (e.g., getting coffee and toast every Sunday) should be possible
b. EarlyBird should deliver all kinds of meals all day, not just breakfast
c. Deliveries should be trackable online by the customer

**Key Focus:** Separate stable business logic from volatile technology concerns.

**Prerequisite:** Complete Mars02 first - small application core principles apply to large systems.
